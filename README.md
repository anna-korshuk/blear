# Blear

A simple jQuery plugin to recreate the iOS7 style blurry header.

    $("#main-header").blear({
        // Options
        radius:       "15", // Blur radius
        background:   "#ffffff",
        opacity:      "60",
        wrapperClass: ".blear-wrap",
        blurClass:    ".blear"
    });

View demo [byjaden.github.com/blear](http://byjaden.github.com/blear)
