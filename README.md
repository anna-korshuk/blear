# Blear

A simple jQuery plugin to recreate the iOS7 style blurry header.

```javascript
$("#main-header").blear({
    // Options
    radius:       "15", // Blur radius
    background:   "#ffffff",
    opacity:      "60",
    wrapperClass: ".blear-wrap",
    blurClass:    ".blear"
});
```

View demo [jaden.io/blear](http://jaden.io/blear) (*works best in Safari*)
